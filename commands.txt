-windows command-

-cls-מנקה את שולחן העבודה
-dir-מראה את כל הקבצים או התקיות שנמצאים באותה תקייה שפתחת
-cd..-מחזיר אותך לאבא של התקייה זאת אומרת תקייה אחת אחורה
-cd name-כניסה לתקייה מסויימת בתוך כונן C 
-md name- יוצר תיקייה בתוך כונן C
-tree-מראה בצורה גרפית את השורשים של התקיות
-cd\ name-מקפיץ לתקייה אחרת שבכונן C
-dir\?-help dir-פותח מסמך עם פקודות
-help rm(remov)-מסביר לך איך למחוק
-rm name /s name-לרשום מה אתה רוצה למחוק 
-dir>name.txt-פותח קובץ ומכניס אליו את התכולה
-copy name.txt c:\-מעתיק קובץ או מסמסך למקום אחר בתקייה
-del name.txt-כדי למחוק קובץ מסויים
-move name.txt c:\-להעביר מקום
-type name.txt-מראה לך מה יש באותו המסמך
-attrib name.txt-מראה אופיין של הקובץ
-attrib /?-מראה איך משנים אופיין של הקובץ
-attrib +R-הופך את הקובץ רק לקריאה
-attrib -R -מחזיר אותו למצב שאפשר לכתוב בוא
-attrib +H -הופך את הקובץ למוסתר
-attrib -H-מחזיר את הקובץ לגלוי
-dir /AH-מראה קבצים מוסתרים
-dir name.txt-מידע על המסמך
-dir *.txt-נותן קבצים שהסיומת שלהם txt
-dir number*.txt-כל הקבצים שמתחילים באותו מספר
-dir number*.txt>name.doc-יוצר תקייה עם כל המסמכים
-dir number*.*-נותן את כל הקבצים שמחילים באותו מספר
dir number?number*.*-נותן את כל הקבצים שיש בהם את המספר הראשון או השלישי וככל שנוסיף סימני שאלה ככה זה יגדל
-notepad-פותח מסמך טקסט(notepad)
